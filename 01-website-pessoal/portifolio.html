Abaixo destaquei meus principais projetos.

Projetos realizados

App-car-shop

Aplicação desenvolvida para disponibilizar uma API com funcionalidades de criar, ler, modificar e deletar informações relativas a venda de carros e motos.

Tecnologias empregadas: Node.js, TypeScript, MongoDB, Mocha e Docker.

App-paises-por-regiao

Aplicação desenvolvida com o objetivo de proporcionar ao usuário conhecer os detalhes dos países selecionados, de acordo com a região de cada continente.

Tecnologias empregadas: HTML, CSS, JavaScript e React.js.

Api-drone-feeder

Aplicação desenvolvida para disponibilizar uma API com funcionalidades de criar, ler, modificar e deletar informações relativas a entregas de pacotes e encomendas realizadas por drones, incluindo upload e download de vídeos.

Tecnologias empregadas: Java, Spring Boot, MySQL, JUnit5 e Docker.
